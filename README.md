## 📚 Category Descriptions

### 🔹 Basics
Simple Python programs covering fundamental concepts like loops, conditions, and arithmetic operations. Programs include factorial, prime number check, Fibonacci series, palindrome check, Armstrong number, even/odd, multiplication table, largest of three numbers, sum of digits, and simple interest calculation.

### 🔹 Functions
Same basic programs but written using **functions**. This demonstrates how to write reusable code using `def`, `return`, and function calls in Python.

### 🔹 OOP (Object-Oriented Programming)
Programs that use **classes and objects** to solve problems. Covers basic OOP concepts like constructors (`__init__`), methods, and object creation. Includes Rectangle, Employee, Calculator, Student, and Interest classes.

### 🔹 Builtins
Demonstrations of Python's **built-in functions** with examples. Covers `print()`, `input()`, `len()`, `type()`, type conversions, `max()`, `min()`, `sum()`, `map()`, `filter()`, `zip()`, `enumerate()`, `any()`, `all()`, `pow()`, and `divmod()`.

## ▶️ How to Run

1. Make sure Python is installed on your system.
2. Open terminal or command prompt.
3. Navigate to the program folder.
4. Run any program using:

```bash
python filename.py
```

**Example:**
```bash
cd Basics
python factorial.py
```
